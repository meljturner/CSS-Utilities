CSS Utility Classes

Colors

This page contains a color utility (_gamut.scss) that explicity defines a primary and secondary hue. The utility auto-generates the complement (cm) and two triadic shades (t1, t2) as well as a light and dark variant of each. These different colors are shown here:

Color Complement Triadic-1 Triadic-2
Primary
Secondary

Note that this paragraph and the previous sub-heading each specify class="text primary cm" so are displayed in the primary complement color.

This paragraph has class="text secondary", and the color changes accordingly. The following paragraph is a longer block with class="background secondary t2" to get a background of the secondary triadic-2 color.

It should be noted that all these colors are generated dynamically so, for example, the primary hue could be changed by javascript and all the primary shades would change. Also, links like this have dynamic hover states that can invert the current color. Check this out, its a long link that can span lines and still handle hover state appropriately.

In addition to primary and secondary colors standard colors are provided for buttons and error messages named info, danger, warning, and success. They also have light and dark variants.
